Overview

Purpose

Load ARC GIS map, use national park api once to get the JSON, then load those points in every time. Use firebase to store user data, implement search bar, as well as possibly a way to store user photos. User photos is extracuricular though.

Audience

My friends and family who love to explore the national parks, but realistically anyone who wants to use it.

Dynamic elements

So honestly like all of it. This was a project I worked on previously, so the more advanced stuff like api calls, and the json being loaded was already done. But for this project and basic JS ill implement a search bar, as well as a progress bar based on how many parks youve been to.
